# 0123-ternary-logic

Covers DMN 1.5 Section 10.3.2-10.3.4: FEEL ternary logic and null/boolean propagation.

- Ensures correct handling of null, true, false, and ternary logic in FEEL expressions.
- Tests propagation of nulls and correct evaluation of AND, OR, NOT with nulls and booleans.

References:
- DMN 1.5 Section 10.3.2 (FEEL values and null)
- DMN 1.5 Section 10.3.3 (Boolean logic)
- DMN 1.5 Section 10.3.4 (Null propagation)
