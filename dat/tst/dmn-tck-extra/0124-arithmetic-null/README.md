# 0124-arithmetic-null

Covers DMN 1.5 Section 10.3.2.4 and 10.3.4: FEEL arithmetic with nulls and numbers.

- Ensures correct handling of nulls in arithmetic operations (+, -, *, /) with numbers and booleans.
- Tests propagation of nulls and type conversion in FEEL arithmetic.

References:
- DMN 1.5 Section 10.3.2.4 (Arithmetic operators)
- DMN 1.5 Section 10.3.4 (Null propagation)
