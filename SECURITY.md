# Security Policy

If you discover a security issue in ORION â€” Optimized Rule Integration & Operations Native, please report it by creating a **private security advisory** on GitHub:

1. Go to https://github.com/omnixs/orion/security/advisories
2. Click "Report a vulnerability"
3. Fill in the details of the security issue
4. Submit the advisory

Alternatively, you can create a confidential issue at https://github.com/omnixs/orion/issues with the label `security`.

We will acknowledge receipt and work to verify and address the issue.

Please do not report vulnerabilities publicly before we have had a reasonable
opportunity to investigate and release a fix or advisory.
